## 🔑 API Access

### Create API Token:
1. Login to the system
2. Click **"Profile"** in navigation
3. Click **"Create Token"**
4. Give it a name
5. **COPY THE TOKEN** (you won't see it again!)

### Test API:
1. Open `test_api.py`
2. Replace `YOUR_API_TOKEN_HERE` with your token
3. Run: `python test_api.py`

### API Endpoints:
- `POST /api/v1/predict` - Make predictions
- `GET /api/v1/history` - Get history
- `GET /api/v1/statistics` - Get statistics
- `GET /api/v1/prediction/{id}` - Get specific prediction

### API Documentation:
http://localhost:5000/api/docs

## 🔍 SHAP Explanations

When you make a prediction, you'll see:

1. **Prediction Result**
   - Benign or Malignant
   - Confidence percentage
   - Risk assessment

2. **Model Comparison**
   - All 3 models' predictions
   - Confidence scores
   - Agreement level

3. **SHAP Force Plot** 🆕
   - Visual explanation
   - Shows which features push towards Benign/Malignant
   - Red = increases risk, Blue = decreases risk

4. **SHAP Waterfall Plot** 🆕
   - Step-by-step breakdown
   - Each feature's contribution
   - Cumulative effect

5. **Top Contributing Features** 🆕
   - Top 5-10 most important features
   - SHAP values
   - Impact direction (increases/decreases risk)
   - Actual feature values
