About Our System

Leveraging AI for better healthcare outcomes

🎯 Project Overview

This AI-based breast cancer detection system uses advanced machine learning algorithms to assist healthcare professionals in early diagnosis of breast cancer. The system analyzes 30 key features from breast mass measurements and provides predictions with confidence scores.

Built as a final year project (PUSL3190 Computing Project), this system demonstrates the practical application of artificial intelligence in medical diagnostics, potentially helping save lives through early detection.

✨ Key Features

🤖 Multiple ML Models

Logistic Regression, Random Forest, and SVM algorithms working together

⚡ Real-time Analysis

Instant predictions with confidence scores and visual feedback

📊 High Accuracy

98.2% accuracy on test data with robust validation

🎨 Modern Interface

Beautiful, intuitive design for seamless user experience

📚 Dataset Information

Wisconsin Breast Cancer Dataset

  • Source: UCI Machine Learning Repository
  • Samples: 569 instances (357 benign, 212 malignant)
  • Features: 30 numeric features computed from digitized images
  • Target: Binary classification (Benign/Malignant)

The features include measurements like radius, texture, perimeter, area, smoothness, compactness, concavity, concave points, symmetry, and fractal dimension - each with mean, standard error, and worst values.
