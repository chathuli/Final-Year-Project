API Documentation

RESTful API for Breast Cancer Detection System

This API allows you to integrate breast cancer detection capabilities into your applications.

Base URL

http://localhost:5000/api/v1

Response Format

All responses are in JSON format with the following structure:

{
    "success": true,
    "data": {...},
    "message": "Optional message"
}

Authentication

All API endpoints require authentication using an API token.

Getting Your API Token

  1. Log in to your account
  2. Go to Profile & API Tokens
  3. Click "Create Token"
  4. Copy your token (you won't see it again!)

Using Your Token

Include your token in the Authorization header:

Authorization: Bearer YOUR_API_TOKEN
Security: Never share your API token or commit it to version control!

cURL Example

GET /api/v1/history
Get Prediction History

Retrieve past predictions.

Query Parameters
  • limit (optional): Number of records (default: 100)
  • offset (optional): Skip records (default: 0)
Response
{
    "success": true,
    "predictions": [
        {
            "id": 123,
            "timestamp": "2024-01-01T12:00:00",
            "prediction": 1,
            "prediction_label": "Malignant",
            "confidence": 98.5,
            "model_name": "Logistic Regression"
        }
    ],
    "total": 150,
    "limit": 100,
    "offset": 0
}
cURL Example
curl -X GET "http://localhost:5000/api/v1/history?limit=10" \
  -H "Authorization: Bearer YOUR_API_TOKEN"

Error Handling

HTTP Status Codes

Code Meaning
200Success
400Bad Request - Invalid input
401Unauthorized - Invalid or missing token
404Not Found
500Internal Server Error

Error Response Format

{
    "success": false,
    "error": "Error type",
    "message": "Detailed error message"
}

Code Examples

Python

import requests

API_TOKEN = "your_api_token_here"
BASE_URL = "http://localhost:5000/api/v1"

headers = {
    "Authorization": f"Bearer {API_TOKEN}",
    "Content-Type": "application/json"
}

# Make a prediction
features = [17.99, 10.38, 122.8, ...]  # 30 features
response = requests.post(
    f"{BASE_URL}/predict",
    headers=headers,
    json={"features": features}
)

result = response.json()
print(f"Prediction: {result['prediction_label']}")
print(f"Confidence: {result['confidence']}%")

JavaScript (Node.js)

const axios = require('axios');

const API_TOKEN = 'your_api_token_here';
const BASE_URL = 'http://localhost:5000/api/v1';

const headers = {
    'Authorization': `Bearer ${API_TOKEN}`,
    'Content-Type': 'application/json'
};

// Make a prediction
const features = [17.99, 10.38, 122.8, ...];  // 30 features

axios.post(`${BASE_URL}/predict`, { features }, { headers })
    .then(response => {
        console.log('Prediction:', response.data.prediction_label);
        console.log('Confidence:', response.data.confidence + '%');
    })
    .catch(error => {
        console.error('Error:', error.response.data);
    });
